The List component provides a composable structure for displaying key-value pairs, metadata, and structured information with consistent styling.

## Basic usage

```tsx
import { List } from '@raystack/apsara';

function UserInfo() {
  return (
    <List>
      <List.Item>
        <List.Label>Name</List.Label>
        <List.Value>John Doe</List.Value>
      </List.Item>
      <List.Item>
        <List.Label>Email</List.Label>
        <List.Value>john@example.com</List.Value>
      </List.Item>
      <List.Item>
        <List.Label>Status</List.Label>
        <List.Value>Active</List.Value>
      </List.Item>
    </List>
  );
}
```

## Inline editing

Combine with form inputs for inline editing.

```tsx
import { List } from '@raystack/apsara';
import { useState } from 'react';

function EditableList() {
  const [editing, setEditing] = useState(false);
  const [value, setValue] = useState('John Doe');

  return (
    <List>
      <List.Item>
        <List.Label>Name</List.Label>
        <List.Value>
          {editing ? (
            <input
              value={value}
              onChange={e => setValue(e.target.value)}
              onBlur={() => setEditing(false)}
              autoFocus
            />
          ) : (
            <span onClick={() => setEditing(true)}>{value}</span>
          )}
        </List.Value>
      </List.Item>
    </List>
  );
}
```

```tsx
        <List.Label>Name</List.Label>
        <List.Value>John Doe</List.Value>
      </List.Item>
    </List>
  );
}
```

## API reference

### List (root)

<ParamField path="children" type="ReactNode" required>
  List content, typically `List.Header` and `List.Item` elements.
</ParamField>

<ParamField path="maxWidth" type="string | number">
  Maximum width of the list container.
</ParamField>

<ParamField path="className" type="string">
  Additional CSS classes to apply.
</ParamField>

<ParamField path="aria-label" type="string">
  Accessible label for the list.
</ParamField>

### List.Header

<ParamField path="children" type="ReactNode" required>
  Header content, typically text.
</ParamField>

<ParamField path="className" type="string">
  Additional CSS classes to apply.
</ParamField>

### List.Item

<ParamField path="children" type="ReactNode" required>
  Item content, typically `List.Label` and `List.Value`.
</ParamField>

<ParamField path="className" type="string">
  Additional CSS classes to apply.
</ParamField>

### List.Label

<ParamField path="children" type="ReactNode" required>
  Label content.
</ParamField>

<ParamField path="minWidth" type="string">
  Minimum width for the label element.
</ParamField>

<ParamField path="className" type="string">
  Additional CSS classes to apply.
</ParamField>

<ParamField path="style" type="React.CSSProperties">
  Inline styles to apply.
</ParamField>

### List.Value

<ParamField path="children" type="ReactNode" required>
  Value content.
</ParamField>

<ParamField path="className" type="string">
  Additional CSS classes to apply.
</ParamField>
